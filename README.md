## 本项目实现的最终作用是基于JSP在线图书借阅管理系统
## 分为3个角色
### 第1个角色为管理员角色，实现了如下功能：
 - 图书信息管理
 - 查看图书借阅信息
 - 用户信息管理
 - 管理员信息修改
 - 管理员信息管理
 - 管理员登录
### 第2个角色为图书馆员工角色，实现了如下功能：
 - 员工信息修改
 - 员工登录
 - 图书信息管理
 - 查看图书借阅信息
 - 用户信息管理
### 第3个角色为用户角色，实现了如下功能：
 - 修改个人信息
 - 图书借阅
 - 用户登录
